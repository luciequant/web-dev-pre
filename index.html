Web Development

Web development is the process of creating websites and web applications. It involves everything that makes a website look good, work properly, and interact with users.

Front-End Development

Everything that users see and interact with on a website.

Programming Languages

HTML, CSS, JavaScript

Back-End Development

Everything that happens behind the scenes, like storing data and running logic.

Programming Languages

Python, PHP, Node.js, Java, C#

Full-Stack Development

Handles both front-end and back-end, building a complete website or application.

HTML

  • HyperText Markup Language

  • The language we use to create the structure of a website.

  • HTML defines all the pieces of a webpage: headings, paragraphs, images, links, buttons, and even forms where users can type or select information.

CSS

  • Cascading Style Sheets

  • It ontrols how a website looks, including colors, fonts, spacing, layout, buttons, form fields, and animations.

  • Essential for making websites look professional and user-friendly.

  • CSS works together with HTML (structure) and JavaScript (interactivity).

  • With CSS selectors you can style elements by tag name, class, ID, groups of elements, elements inside other elements, or elements in a specific state like hover.

JavaScript

  • JavaScript allows a website to respond to user actions and interact with users.

  • It can make buttons clickable, show or hide content, validate forms, and create animations or smooth transitions on a webpage.

  • Key Concepts:

    • Functions: Reusable blocks of code that perform specific tasks.
    • Events: Actions that trigger functions, such as clicks, typing, or hovering.
